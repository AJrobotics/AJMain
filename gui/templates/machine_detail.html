🔒 External Access Guide (via {{ machine_name }})

{{ machine_name }} serves as the gateway for external access to the home network. Here are the recommended methods:

Option 1: Tailscale VPN (Recommended)
Install Tailscale on {{ machine_name }} and your phone/laptop.
• On {{ machine_name }}: curl -fsSL https://tailscale.com/install.sh | sh && sudo tailscale up
• Enable subnet routing: sudo tailscale up --advertise-routes=192.168.1.0/24
• Then access all machines via Tailscale IP from anywhere
Option 2: SSH Port Forwarding (Router)
• Forward router port (e.g. 2222) → {{ machine_info.get('host', '') }}:22
• From outside: ssh -p 2222 {{ machine_info.get('username', '') }}@YOUR_PUBLIC_IP
• Then hop to other machines: ssh dongchul@192.168.1.91
Option 3: SSH Jump Host
• Direct access to CashCow via Christy as jump:
ssh -J {{ machine_info.get('username', '') }}@YOUR_PUBLIC_IP:2222 dongchul@192.168.1.91
• Add to ~/.ssh/config for convenience
Option 4: Reverse SSH Tunnel
• No router config needed. From {{ machine_name }}:
ssh -R 2222:localhost:22 user@cloud-server
• Then from anywhere: ssh -p 2222 {{ machine_info.get('username', '') }}@cloud-server
